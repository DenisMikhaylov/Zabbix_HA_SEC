1. Устанавливаем zabbix server на второй сервер

```
apt install zabbix-server-mysql zabbix-agent
```

2. Переключаемся на виртуальную машину с первым сервером Zabbix

3. Настройка базы данных mysql 

```
nano /etc/mysql/mariadb.conf.d/50-server.cnf
```
4. Прописать доступ к серверу DB
```
bind-address = 0.0.0.0
```
5. Перезапустить службу DB
```
systemctl restart mariadb
```

6. Добавление прав

```bash
mysql -uroot -p
```
```sql
create user zabbix@<ip address Debian-3> identified by 'password';
```
```sql
grant all privileges on zabbix.* to zabbix@<ip address Debian-3>;
```


7. Настройка первой ноды сервера Zabbix

```
nano /etc/zabbix/zabbix_server.conf
```
8. Добавить строки
```
HANodeName=zabbix-node1
NodeAddress=<ip адрес первого сервера>
```
9. Перезапускаем сервер

```
systemctl restart zabbix-server
```

10. Проверка работы сервера

```
zabbix_server -R ha_status
```
или

На web report-sistem information

11. Настройка второго сервера Zabbix

12. Подключиться к второму серверу для Zabbix

```
nano /etc/zabbix/zabbix_server.conf
```
```
HANodeName=zabbix-node2
NodeAddress=<ip address Debian-3>
DBHost=<ip адрес сервера DB>
DPPassword=password
```
13. Перезапускаем сервер

```
systemctl restart zabbix-server
```
